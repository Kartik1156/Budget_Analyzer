# 💸 Smart Financial Risk Analyzer

This is a simple web app that helps users understand their spending and financial condition.
You can enter your monthly income and expenses, and the app will show how much you’re saving and whether your spending is risky or not.

---

## 🔍 What this project does

* Calculates total expenses and savings
* Shows savings percentage
* Gives a basic risk level (low, medium, high)
* Displays graphs to understand spending
* Predicts next month’s financial condition

---

## 🧠 Concepts used

This project is based on topics from Probability and Statistics:

* Basic data analysis
* Ratio-based risk calculation
* Data visualization (graphs)
* Simple decision rules

## ▶️ How to run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the app:

```
streamlit run app.py
```
